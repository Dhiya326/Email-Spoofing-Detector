# Email Spoofing Detector

A frontend web application that analyzes email addresses and email headers to identify potential spoofing attempts.

## Features

- Email format validation (Regex)
- Domain comparison
- Free email provider detection
- Suspicious keyword detection
- Risk score calculation
- Email header analysis
- Detection history (Local Storage ready)
- Report export (TXT)
- Responsive UI
- Modular JavaScript architecture

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## How to Run

1. Download the project.
2. Open the folder in VS Code.
3. Install the Live Server extension (optional).
4. Open `index.html` in a browser or run with Live Server.
